# 飞书通知配置方法



## 机器人配置

### 1. 注册飞书账号

认准官网：https://www.feishu.cn/。别注册到国际版了，以下省略。

### 2. 建立机器人

1. 打开飞书开放平台https://open.feishu.cn/app?lang=zh-CN

2. 在**企业自建应用**下点击**创建企业自建应用**

   > 一定要是**企业自建应用**， 商店应用用不了

3. 选择当前应用，在**左边**菜单选择**添加应用能力**添加**机器人**能力（点击机器人卡片下的添加）

4. **左边**菜单选择**权限管理**，选择**批量导入/导出权限**，粘贴以下内容后点击**下一步，确认新增权限**。

```json
{
  "scopes": {
    "tenant": [
      "contact:user.id:readonly",
      "im:message",
      "im:resource"
    ],
    "user": []
  }
}
```

5. 在页面最上方应用右边点击**创建版本**，填写必填内容后点击**保存**。然后点击**确认发布**。

## Komari配置

`Webhook provider` 选择 `Feishu`

### App ID与App Secret

最左侧菜单选择**凭证与基础信息**。

`App ID` 和 `App Secret` 就在页面上方

### User Mobile

填你飞书注册的手机号
